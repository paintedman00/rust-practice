# Stack Calculator

A simple Reverse Polish Notation (RPN) calculator implemented in Rust.

## Usage

Run the program and enter expressions in RPN format, separated by spaces. The calculator will evaluate the expression and print the result.

Example:

```
> 5 2 +
7
> 10 3 - 4 *
28
```

Supported operators: `+`, `-`, `*`, `/`

Type `exit` to quit.
