# Text Stats

A simple command-line tool to calculate text statistics, similar to `wc`.

## Features

*   Counts lines, words, and characters from input.
*   Reads from standard input or a specified file.

## Usage

```bash
text_stats <file>
text_stats # Reads from standard input
```

## Example

```bash
echo "Hello world!\nThis is a test." | text_stats
```
