# String Hashmap Demo

A simple command-line tool that demonstrates basic hashmap operations in Rust.

## Usage

```bash
cargo run
```

The program will:

1.  Create a `HashMap` to store string keys and integer values.
2.  Insert some key-value pairs.
3.  Retrieve a value based on a key.
4.  Check if a key exists.
5.  Remove a key-value pair.

## Dependencies

None.
