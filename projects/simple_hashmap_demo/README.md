# Simple Hash Map Demo

A simple command-line interface demonstrating the usage of Rust's `HashMap`.

## Usage

Run the program:

```bash
cargo run
```

The program will prompt you to enter keys and values to store in the hash map.  It will then allow you to retrieve values based on the entered keys.

## Example

```
Enter key: name
Enter value: Alice
Enter key: age
Enter value: 30

Enter key to retrieve: name
Value for key 'name': Alice

Enter key to retrieve: age
Value for key 'age': 30

Enter key to retrieve: city
Key 'city' not found.
```
