# Hash Map Demo

A simple command-line application demonstrating basic Hash Map operations in Rust.

## Usage

```bash
cargo run
```

The program will:

1.  Create a hash map.
2.  Insert some key-value pairs.
3.  Retrieve values by key.
4.  Remove a key-value pair.
5.  Iterate over the hash map.

## Building

Make sure you have Rust installed. Then:

```bash
cargo build
```

## Running

```bash
cargo run
```
