# Binary Transmuter

A simple command-line tool to convert binary numbers to decimal.

## Usage

```bash
binary_transmuter 101101
```

This will output the decimal equivalent of the binary number `101101`.

## Installation

1.  Make sure you have Rust installed. If not, install it from [https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install).
2.  Clone this repository.
3.  Navigate to the project directory.
4.  Build the project using `cargo build --release`.
5.  The executable will be located in `target/release/binary_transmuter`.
