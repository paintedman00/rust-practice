# Text Navigator

This is a simple command-line tool for searching text patterns within files, similar to `grep`.

## Usage

```
text_navigator <pattern> <file>
```

*   `<pattern>`: The search pattern (a string).
*   `<file>`: The file to search within.

## Example

To search for the word "example" in the file "my_file.txt":

```
text_navigator example my_file.txt
```

## Features

*   Basic pattern matching.
*   Prints the lines containing the matched pattern.
