# bin_dec

A simple command-line tool to convert binary numbers to decimal numbers.

## Usage

```bash
bin_dec 1010
```

This will output the decimal representation of the binary number 1010.

## Installation

Make sure you have Rust installed. You can install it from [https://www.rust-lang.org/](https://www.rust-lang.org/)

Clone this repository:

```bash
git clone <repository_url>
cd bin_dec
```

Build and run the project:

```bash
cargo run -- <binary_number>
```
