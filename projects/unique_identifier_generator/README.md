# Unique Identifier Generator

A simple command-line tool to generate UUIDs (Universally Unique Identifiers).

## Usage

Run the program from the command line to generate a UUID:

```bash
cargo run
```

## Building

To build the project, use the following command:

```bash
cargo build
```

## Installation

To install the project, use the following command:

```bash
cargo install --path .
```
