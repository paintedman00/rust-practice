# Linked List Lab

A simple command-line tool to experiment with linked list operations.

## Usage

Run the program with `cargo run` to see a demonstration of linked list creation and traversal.

## Features

*   Creates a linked list.
*   Traverses and prints the linked list.

## Future Enhancements

*   Implement insertion, deletion, and searching operations.
*   Add command-line arguments to control list creation.
